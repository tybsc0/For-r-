IV) Write a program to input the Number of calls made by the user. Calculate and display the bill amount, which includes a fixed rent of rs. 400. The first 150 calls are free with excess calls charged at 80 paise each.

import java.util.*;
class callbill
{
 public static void main(String args[])
	{
	 Scanner s = new Scanner(System.in);
	 int n; 
	 float amt;
	 System.out.println("Enter the no. of Calls");
	 n = s.nextInt();
	 if(n<=150)
	 amt = 400;
	 else
	 amt = 400+((n-150)*0.8f);
	 System.out.println("The Bill amount of the user is "+amt);
	}
}

VIII) Write a program to create a two dimensional array of size [5] x [3]. Display the transpose of the matrix.

import java.util.*;
class transpose
{
 public static void main(String args[])
	{
	 int x[][] = new int[5][3];
	 int y[][] = new int[3][5];
	 int r, c;
	 Scanner s = new Scanner(System.in);
	 for(r=0; r<x.length; r++)
		{
		 System.out.println("Enter Data for row "+r);
		 for(c=0; c<3; c++)
		 x[r][c] = s.nextInt();
		}
	 System.out.println("Matrix X is -");
	 for(r=0; r<x.length; r++)
		{
		 for(c=0; c<3; c++)
			{
	 		 System.out.print(x[r][c]+" ");
			}
		 System.out.print("\n");
		}
	 System.out.println("The Transpose of matrix X is -");
	 for(r=0; r<y.length; r++)
		{
		 for(c=0; c<5; c++)
			{
			 y[r][c] = x[c][r];
			 System.out.print(y[r][c]+" ");
			}
		 System.out.print("\n");
		}
	}
}

X) Write a program to display the first n Fibonacii numbers. Value of “n” should be taken from user.

import java.util.*;
class fibonacii
{
 public static void main(String args[])
	{
	 Scanner s = new Scanner(System.in);
	 int n, x=1, y=-1, sum;
	 System.out.println("Enter a no.");
	 n = s.nextInt();
	 System.out.println("The Fibonacii series is ");
	 for(int i=1; i<=n; i++)
		{
		 sum = x+y;
		 System.out.print(sum+" ");
		 y = x;
		 x = sum;
		}
	}
}
